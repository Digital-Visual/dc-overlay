## Installation

`CDN`

```html
<!--Overlay Package-->
<script src="libs/dc-overlay/dc.overlay.min.js"></script>
```

`NPM / YARN`

```shell
   yarn add @dvgis/dc-overlay
   npm install  @dvgis/dc-overlay
```

```js
import DcOverlay from  'dvgis/dc-overlay/dist/dc.overlay.min' //Overlay Packaget
```

## Start

```js
DC.use(DcOverlay)
```
